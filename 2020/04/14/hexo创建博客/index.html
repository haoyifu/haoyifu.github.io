    使用hexo搭建博客，部署到github和coding

    下载git软件

    官网：https://git-scm.com/

    下载node.js

    官网：https://nodejs.org/zh-cn/download/

    本人使用的是windows 10所以下的是windows版本

    这两个软件安装没什么技术含量，一路下一步就可以(可以更改目录)。

    注册账号

    分别在coding和github上创建账号，用户名建议都一样

    coding官网：https://coding.net/

    github官网：https://github.com/

    注册成功截图

    安装hexo

    hexo官网：https://hexo.io/zh-cn/

    由于官网上推荐的方法有点慢

    故采用其他的按爪给你方式

    打开git

    更新源:

    npm install -g cnpm –registry=https://registry.npm.taobao.org

    安装:cnpm install hexo-cli -g

    初始化博客

    不要hexo init太慢

    过程在这：https://github.com/kjhuanhao/HEXO-HHC-BLOG

    进入你要创建博客的文件夹中git bash 或者打开终端

    执行：git clone https://github.com/kjhuanhao/HEXO-HHC-BLOG.git

    如果不成功多执行几次

    你会得到一个文件夹，进入该文件夹

    cd HEXO-HHC-BLOG/

    然后更新

    cnpm install

    查看博客

    hexo s

    打开浏览器访问

    http://localhost:4000/

    用git生成公钥，并配置到coding和github上。

    生成公钥

    可以参考https://help.coding.net/docs/project/features/ssh.html#pageTitle

    生成公钥（生成中可以一直回车）

    验证：ssh -T git@e.coding.net

    再次复制密钥粘贴到github上

    验证
